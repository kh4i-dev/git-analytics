Git Diff Workflow

Để AI gợi ý commit và review chính xác nhất, bạn cần cung cấp mã diff tiêu chuẩn từ terminal của dự án:

  • Chưa stage (chưa git add): git diff
  • Đã stage (đã chạy git add): git diff --cached
  • Commit gần nhất: git show HEAD
